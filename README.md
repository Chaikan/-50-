#  Internet Security Website

เว็บไซต์ให้ความรู้เกี่ยวกับความปลอดภัยบนอินเทอร์เน็ต ครอบคลุมเรื่อง Phishing, Password Security และ Virus พร้อมฟังก์ชัน Interactive เพื่อให้ผู้ใช้เรียนรู้ผ่านการทดลองจริง

##  เนื้อหาหลักของเว็บไซต์
-  **Phishing** — ความหมาย รูปแบบ และตัวอย่างจำลองการหลอกลวง
-  **Password Security** — การตั้งรหัสผ่านที่ปลอดภัย พร้อมระบบวัดความแข็งแรงของรหัสผ่าน
-  **Virus** — ประเภทไวรัส ผลกระทบ วิธีป้องกัน พร้อมแบบทดสอบความรู้
-  **Member** — รายชื่อสมาชิกทีม พร้อม popup รายละเอียด
-  **Contact** — ฟอร์มสำหรับส่งข้อความติดต่อ

##  ฟีเจอร์เด่น
- หน้าเว็บแบบ Responsive รองรับมือถือและคอมพิวเตอร์
- เมนู Dropdown และ Hamburger Menu
- Popup / Modal แสดงข้อมูลสมาชิก
- ตัวอย่างจำลอง Phishing แบบ Interactive
- Password Strength Checker แบบเรียลไทม์
- Quiz ทดสอบความรู้เรื่องไวรัส

## 🛠️ เทคโนโลยีที่ใช้
- HTML5
- CSS3
- JavaScript (Vanilla JS)

##  โครงสร้างไฟล์
```
project/
│
├── index.html          # หน้าแรก
├── phishing.html       # เนื้อหา Phishing + Demo
├── password.html       # เนื้อหา Password + Strength Checker
├── virus.html          # เนื้อหา Virus + Quiz
├── member.html         # รายชื่อสมาชิก + Modal
├── contact.html        # ฟอร์มติดต่อ
├── style.css           # ไฟล์ CSS กลาง
└── pic/                # รูปสมาชิก
```

##  วิธีใช้งาน
1. ดาวน์โหลดหรือ clone โปรเจกต์
2. เปิดไฟล์ `index.html` ผ่านเว็บเบราว์เซอร์

หรือถ้าใช้ Live Server (แนะนำ):
```
เปิด index.html ผ่าน Live Server
```

##  เป้าหมายของโปรเจกต์
เพื่อให้ผู้ใช้งานเข้าใจภัยคุกคามบนอินเทอร์เน็ต และสามารถป้องกันตัวเองได้อย่างถูกต้อง
